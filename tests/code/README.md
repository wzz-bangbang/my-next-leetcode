# 代码题模块测试用例

> 测试范围：代码题列表、详情、代码编辑器

---

## 测试环境准备

- [ ] 数据库已启动
- [ ] 代码题数据已初始化（`code_categories`、`code_questions`）

---

## 1. 获取题目列表

### TC-CODE-LIST-001 正常获取列表
| 项目 | 内容 |
|------|------|
| **操作步骤** | 访问代码题页面 / 调用 `GET /api/code` |
| **预期结果** | 返回所有代码题列表 |
| **检查点** | 每个题目有 id, slug, title, tags, difficulty |

### TC-CODE-LIST-002 难度筛选
| 项目 | 内容 |
|------|------|
| **操作步骤** | 点击难度筛选按钮 |
| **预期结果** | 只显示对应难度的题目 |
| **检查点** | 简单=1, 中等=2, 困难=3 |

### TC-CODE-LIST-003 分类筛选
| 项目 | 内容 |
|------|------|
| **操作步骤** | 点击分类筛选 |
| **预期结果** | 只显示该分类下的题目 |

### TC-CODE-LIST-004 无数据情况
| 项目 | 内容 |
|------|------|
| **前置条件** | 数据库无代码题数据 |
| **预期结果** | 返回空数组 `[]` |

## 测试数据

```sql
-- 查看代码题（前10条）
SELECT id, slug, title, difficulty FROM code_questions LIMIT 10;

-- 查看用户进度
SELECT * FROM user_code_progress WHERE user_id = 1;

-- 查看用户保存的代码
SELECT * FROM user_answers WHERE user_id = 1;
```
