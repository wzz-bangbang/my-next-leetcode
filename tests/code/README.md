## 6. 运行代码

### TC-CODE-RUN-001 正常运行

| 项目         | 内容                              |
| ------------ | --------------------------------- |
| **操作步骤** | 编写代码后点击"执行代码"          |
| **预期结果** | Console 区域显示 console.log 输出 |

### TC-CODE-RUN-002 运行错误代码

| 项目         | 内容                   |
| ------------ | ---------------------- |
| **测试数据** | 含语法错误的代码       |
| **预期结果** | Error 区域显示错误信息 |

**具体测试用例**：

```javascript
// 用例1：语法错误 - 缺少括号
function test( {
  return 1;
}
// 预期：SyntaxError: Unexpected token '{'

// 用例2：引用未定义变量
console.log(undefinedVar);
// 预期：ReferenceError: undefinedVar is not defined

// 用例3：类型错误 - 调用非函数
const x = 123;
x();
// 预期：TypeError: x is not a function

// 用例4：JSON 解析错误
JSON.parse('{invalid}');
// 预期：SyntaxError: Unexpected token 'i'

// 用例5：访问 null 的属性
const obj = null;
console.log(obj.name);
// 预期：TypeError: Cannot read properties of null
```
