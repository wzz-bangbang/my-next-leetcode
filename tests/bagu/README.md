# 八股文模块测试用例

> 测试范围：八股题列表、详情、分类

---

## 测试环境准备

- [ ] 数据库已启动
- [ ] 八股题数据已初始化（`bagu_categories`、`bagu_questions`）

---

## 1. 获取分类和题目列表

| 项目 | 内容 |
|------|------|

| **检查点** | 1. categories 数组非空<br>2. 每个分类有 id, slug, name, questions<br>3. 每个题目有 id, slug, title, hasAnswer |

| 项目 | 内容 |
|------|------|
| **预期结果** | 题目按 sort_order 排序 |

### TC-BAGU-LIST-004 无数据情况
