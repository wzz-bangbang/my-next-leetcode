# 用户答案模块测试用例

> 测试范围：代码保存、获取

---

## 测试环境准备

- [ ] 数据库已启动
- [ ] 测试用户已登录

---

## 1. 获取用户答案

### TC-ANS-GET-001 获取所有已答题目ID
| 项目 | 内容 |
|------|------|
| **前置条件** | 用户已登录，保存过若干题目代码 |
| **操作步骤** | 调用 `GET /api/answers` |
| **预期结果** | 返回 `{ answeredIds: [1, 3, 5] }` |

### TC-ANS-GET-002 获取指定题目答案
| 项目 | 内容 |
|------|------|
| **前置条件** | 用户保存过题目1的代码 |
| **操作步骤** | 调用 `GET /api/answers?questionId=1` |
| **预期结果** | 返回 `{ code: "function..." }` |

### TC-ANS-GET-003 获取未保存的题目
| 项目 | 内容 |
|------|------|
| **前置条件** | 用户未保存过该题 |
| **预期结果** | 返回 `{ code: "" }` |

### TC-ANS-GET-004 未登录获取
| 项目 | 内容 |
|------|------|
| **前置条件** | 用户未登录 |
| **预期结果** | 状态码 401 |

---

## 2. 保存用户答案

### TC-ANS-SAVE-001 新建答案
| 项目 | 内容 |
|------|------|
| **前置条件** | 该题无保存记录 |
| **操作步骤** | `POST /api/answers` `{ questionId: 99, code: "..." }` |
| **预期结果** | 成功，数据库插入新记录 |

### TC-ANS-SAVE-002 更新已有答案
| 项目 | 内容 |
|------|------|
| **前置条件** | 该题有保存记录 |
| **操作步骤** | 发送新的 code |
| **预期结果** | 成功，数据库更新 |

### TC-ANS-SAVE-003 保存空代码
| 项目 | 内容 |
|------|------|
| **测试数据** | `code: ""` |
| **预期结果** | 前端拦截，提示"代码内容不能为空" |
| **备注** | 后端允许空代码，但前端 `handleSave` 会拦截 |

### TC-ANS-SAVE-004 缺少参数
| 项目 | 内容 |
|------|------|
| **测试数据** | 缺少 code 字段 |
| **预期结果** | 状态码 400 |

### TC-ANS-SAVE-005 未登录保存
| 项目 | 内容 |
|------|------|
| **前置条件** | 用户未登录 |
| **预期结果** | 状态码 401 |

## 3. 代码长度限制

### TC-ANS-LIMIT-001 正常长度代码
| 项目 | 内容 |
|------|------|
| **测试数据** | 100行，2000字符 |
| **预期结果** | 保存成功 |

### TC-ANS-LIMIT-002 超过400行
| 项目 | 内容 |
|------|------|
| **测试数据** | 401行代码 |
| **预期结果** | 前端拦截保存，提示超出限制 |
| **备注** | 代码可正常输入和执行，仅保存时拦截；红色警告已在编辑器显示 |

| **备注** | 代码可正常输入和执行，仅保存时拦截；红色警告已在编辑器显示 |

## 5. 前端交互

### TC-ANS-UI-001 编辑器加载保存的代码
| 项目 | 内容 |
|------|------|
| **前置条件** | 用户保存过该题代码 |
| **操作步骤** | 打开题目详情 |
| **预期结果** | 编辑器显示保存的代码，而非模板 |

### TC-ANS-UI-002 切换题目时自动保存
| 项目 | 内容 |
|------|------|
| **前置条件** | 当前题目代码已修改（与模板不同） |
| **操作步骤** | 点击侧边栏切换到其他题目 |
| **预期结果** | 自动保存当前代码到数据库 |
| **备注** | 仅在代码通过长度校验时保存 |

### TC-ANS-UI-003 切回题目加载已保存代码
| 项目 | 内容 |
|------|------|
| **前置条件** | 题目 A 已保存代码 |
| **操作步骤** | 从题目 B 切换回题目 A |
| **预期结果** | 加载题目 A 最新保存的代码 |
| **备注** | 每次切换都从后端获取，保证多设备数据一致 |

### TC-ANS-UI-004 状态栏显示代码统计
| 项目 | 内容 |
|------|------|
| **操作步骤** | 查看编辑器底部状态栏 |
| **预期结果** | 显示行数和字符数 |

### TC-ANS-UI-005 超限时状态栏警告
| 项目 | 内容 |
|------|------|
| **操作步骤** | 输入超限代码 |
| **预期结果** | 状态栏显示红色警告 |

## 测试数据

```sql
-- 查看用户保存的代码
SELECT * FROM user_answers WHERE user_id = 1;

-- 查看代码长度
SELECT question_id, LENGTH(code) as length, 
       (LENGTH(code) - LENGTH(REPLACE(code, '\n', ''))) as lines 
FROM user_answers WHERE user_id = 1;

-- 清理测试数据
DELETE FROM user_answers WHERE user_id = 1;
```
