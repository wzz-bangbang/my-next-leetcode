# 收藏模块测试用例

> 测试范围：收藏的增删查、同步

---

## 测试环境准备

- [ ] 数据库已启动
- [ ] 测试用户已登录

---

## 1. 获取收藏列表

### TC-FAV-GET-001 获取所有收藏

| 项目         | 内容                                |
| ------------ | ----------------------------------- |
| **前置条件** | 用户已登录，有若干收藏              |
| **操作步骤** | 调用 `GET /api/favorites`           |
| **预期结果** | 返回 `{ bagu: [...], code: [...] }` |

### TC-FAV-GET-002 获取代码题收藏

| 项目         | 内容                                |
| ------------ | ----------------------------------- |
| **操作步骤** | 调用 `GET /api/favorites?type=code` |
| **预期结果** | 返回 `{ ids: [...] }` 只含代码题ID  |

### TC-FAV-GET-003 获取八股题收藏

| 项目         | 内容                                |
| ------------ | ----------------------------------- |
| **操作步骤** | 调用 `GET /api/favorites?type=bagu` |
| **预期结果** | 返回 `{ ids: [...] }` 只含八股题ID  |

### TC-FAV-GET-004 无收藏

| 项目         | 内容                          |
| ------------ | ----------------------------- |
| **前置条件** | 用户无任何收藏                |
| **预期结果** | 返回 `{ bagu: [], code: [] }` |

### TC-FAV-GET-005 未登录获取

| 项目         | 内容       |
| ------------ | ---------- |
| **前置条件** | 用户未登录 |
| **预期结果** | 状态码 401 |

## 2. 切换单个收藏状态

### TC-FAV-TOGGLE-001 添加收藏

| 项目         | 内容                                                                       |
| ------------ | -------------------------------------------------------------------------- |
| **前置条件** | 题目未收藏                                                                 |
| **操作步骤** | `PATCH /api/favorites` `{ type: "code", questionId: 1, isFavorite: true }` |
| **预期结果** | 返回 `{ success: true }`，数据库更新                                       |

### TC-FAV-TOGGLE-002 取消收藏

| 项目         | 内容                                                                        |
| ------------ | --------------------------------------------------------------------------- |
| **前置条件** | 题目已收藏                                                                  |
| **操作步骤** | `PATCH /api/favorites` `{ type: "code", questionId: 1, isFavorite: false }` |
| **预期结果** | 返回 `{ success: true }`，数据库更新                                        |

### TC-FAV-TOGGLE-003 重复添加收藏

| 项目         | 内容                        |
| ------------ | --------------------------- |
| **前置条件** | 题目已收藏                  |
| **操作步骤** | 再次发送 `isFavorite: true` |
| **预期结果** | 成功，无报错（幂等）        |

### TC-FAV-TOGGLE-004 缺少参数

| 项目         | 内容            |
| ------------ | --------------- |
| **测试数据** | 缺少 questionId |
| **预期结果** | 状态码 400      |

### TC-FAV-TOGGLE-005 无效 type

| 项目         | 内容              |
| ------------ | ----------------- |
| **测试数据** | `type: "invalid"` |
| **预期结果** | 状态码 400        |

### TC-FAV-TOGGLE-006 未登录操作

| 项目         | 内容       |
| ------------ | ---------- |
| **前置条件** | 用户未登录 |
| **预期结果** | 状态码 401 |

### TC-FAV-TOGGLE-007 无效 questionId（负数）

| 项目         | 内容                                                 |
| ------------ | ---------------------------------------------------- |
| **测试数据** | `{ type: "code", questionId: -1, isFavorite: true }` |
| **预期结果** | 状态码 400，返回 `Invalid question ID`               |

### TC-FAV-TOGGLE-008 不存在的 questionId

| 项目         | 内容                                                     |
| ------------ | -------------------------------------------------------- |
| **测试数据** | `{ type: "code", questionId: 999999, isFavorite: true }` |
| **预期结果** | 状态码 404，返回 `Question not found`                    |

## 测试数据

```sql
-- 查看用户收藏
SELECT * FROM user_code_progress WHERE user_id = 1 AND is_favorite = 1;
SELECT * FROM user_bagu_progress WHERE user_id = 1 AND is_favorite = 1;

-- 清理测试数据
UPDATE user_code_progress SET is_favorite = 0 WHERE user_id = 1;
UPDATE user_bagu_progress SET is_favorite = 0 WHERE user_id = 1;
```
