# 认证模块测试用例

> 测试范围：用户注册、登录、密码管理、邮箱管理、账号注销

---

## 测试环境准备

- [ ] 数据库已启动（Docker MySQL）
- [ ] 邮件服务已配置（Resend）
- [ ] 清空测试数据：`email_verification_codes`、测试用户

## 1. 邮箱密码注册

### TC-REG-001 正常注册

| 项目         | 内容                                                                               |
| ------------ | ---------------------------------------------------------------------------------- |
| **前置条件** | 邮箱未被注册                                                                       |
| **操作步骤** | 1. 点击登录按钮<br>2. 切换到"注册"页签<br>3. 输入邮箱、密码、用户名<br>4. 点击注册 |
| **测试数据** | 邮箱：`test@example.com`<br>密码：`Test123!@#`<br>用户名：`测试用户`               |
| **预期结果** | 提示"注册成功"，弹窗切换到登录页签                                                 |
| **检查点**   | 数据库 users 表有新记录，password_hash 已加密                                      |

### TC-REG-002 邮箱已注册

| 项目         | 内容                           |
| ------------ | ------------------------------ |
| **前置条件** | 邮箱已被注册                   |
| **操作步骤** | 同 TC-REG-001                  |
| **预期结果** | 提示"该邮箱已注册"，状态码 409 |

### TC-REG-003 邮箱格式错误

| 项目         | 内容                       |
| ------------ | -------------------------- |
| **测试数据** | 邮箱：`invalid-email`      |
| **预期结果** | 提示"请输入有效的邮箱地址" |

### TC-REG-004 密码强度不足

| 项目         | 内容                                             |
| ------------ | ------------------------------------------------ |
| **测试数据** | 密码：`123456`（纯数字）                         |
| **预期结果** | 提示密码需包含大写、小写、数字、符号中的至少三种 |

### TC-REG-005 密码长度不足

| 项目         | 内容                     |
| ------------ | ------------------------ |
| **测试数据** | 密码：`Aa1!`（少于8位）  |
| **预期结果** | 提示"密码长度需为8-14位" |

### TC-REG-006 密码长度超限

| 项目         | 内容                                 |
| ------------ | ------------------------------------ |
| **测试数据** | 密码：`Test123!@#456789`（超过14位） |
| **预期结果** | 提示"密码长度需为8-14位"             |

### TC-REG-007 昵称长度限制

| 项目         | 内容                                   |
| ------------ | -------------------------------------- |
| **操作步骤** | 在昵称输入框尝试输入超过20个字符       |
| **预期结果** | 输入框限制最多20字符，超出部分无法输入 |

### TC-REG-008 用户名为空

| 项目         | 内容                           |
| ------------ | ------------------------------ |
| **测试数据** | 用户名留空                     |
| **预期结果** | 注册成功，用户名默认取邮箱前缀 |
