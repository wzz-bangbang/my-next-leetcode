# POST /api/auth/register - 用户注册

**鉴权**：无需登录

## 请求参数

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `email` | string | 是 | 邮箱地址 |
| `password` | string | 是 | 密码（8-14位，需包含大写、小写、数字、符号中的至少三种） |
| `name` | string | 否 | 用户名（最大20字符，不填则取邮箱前缀） |

## 请求示例

```json
{
  "email": "user@example.com",
  "password": "MyPassword123!",
  "name": "张三"
}
```

## 成功返回

```json
{
  "success": true,
  "message": "注册成功"
}
```

## 错误情况

| 状态码 | 说明 |
|--------|------|
| 400 | 邮箱/密码为空 / 邮箱格式错误 / 密码强度不足 / 用户名过长 |
| 409 | 该邮箱已注册 |
| 500 | 注册失败 |
