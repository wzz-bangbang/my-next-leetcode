# POST /api/auth/change-password - 修改密码

**鉴权**：需要登录

## 请求参数

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `oldPassword` | string | 是 | 原密码 |
| `newPassword` | string | 是 | 新密码（8-14位，需包含大写、小写、数字、符号中的至少三种） |

## 请求示例

```json
{
  "oldPassword": "OldPassword123!",
  "newPassword": "NewPassword456!"
}
```

## 成功返回

```json
{
  "message": "密码修改成功"
}
```

## 错误情况

| 状态码 | 说明 |
|--------|------|
| 400 | 原密码为空 / 新密码强度不足 / 第三方登录用户无密码 |
| 401 | 未登录 / 原密码错误 |
| 404 | 用户不存在 |
| 500 | 服务器错误 |

## 注意

第三方登录（GitHub/Google）用户没有密码，调用此接口会返回 400 错误。
