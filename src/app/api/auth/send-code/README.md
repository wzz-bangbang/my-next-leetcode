# POST /api/auth/send-code - 发送验证码

**鉴权**：无需登录

## 请求参数

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `email` | string | 是 | 邮箱地址 |
| `type` | number | 否 | 验证码类型：1=登录（默认），2=重置密码，3=换绑邮箱 |

## 请求示例

```json
{
  "email": "user@example.com",
  "type": 1
}
```

## 成功返回

```json
{
  "success": true,
  "message": "验证码已发送"
}
```

## 错误情况

| 状态码 | 说明 |
|--------|------|
| 400 | 邮箱格式错误 / 无效的验证码类型 / 该邮箱未注册（type=2） |
| 429 | 发送频率限制 |
| 500 | 发送失败 |

## 安全限制

| 规则 | 限制 |
|------|------|
| 同邮箱 | 60秒内只能发送1次 |
| 同邮箱 | 10分钟内最多3次 |
| 同IP | 每小时最多10次 |
