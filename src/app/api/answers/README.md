# 用户答案 API

> 路径：`/api/answers`

**注意**：所有接口需要用户登录

---

## GET - 获取用户答案

### 获取所有已答题目ID

**请求**：`GET /api/answers`

**鉴权**：需要登录

**返回字段**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `answeredIds` | number[] | 已有答案的题目ID列表 |

**返回示例**：
```json
{
  "answeredIds": [1, 3, 5, 7]
}
```

---

### 获取指定题目答案

**请求**：`GET /api/answers?questionId={id}`

**参数**：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `questionId` | string | 是 | 题目ID |

**鉴权**：需要登录

**返回字段**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `code` | string | 用户保存的代码（无则为空字符串） |

**返回示例**：
```json
{
  "code": "function promiseAll(promises) {\n  // 用户代码\n}"
}
```

---

## POST - 保存用户答案

**请求**：`POST /api/answers`

**鉴权**：需要登录

**请求体**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `questionId` | number | 是 | 题目ID |
| `code` | string | 是 | 用户代码 |

**请求示例**：
```json
{
  "questionId": 1,
  "code": "function promiseAll(promises) {\n  // 实现\n}"
}
```

**返回字段**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `success` | boolean | 是否成功 |

**返回示例**：
```json
{
  "success": true
}
```

---

## 错误响应

| 状态码 | 说明 |
|--------|------|
| 400 | 参数错误（缺少 questionId 或 code） |
| 401 | 未登录 |
| 500 | 数据库错误 |

---

## 前端限制

| 限制项 | 值 |
|--------|-----|
| 最大行数 | 400 行 |
| 最大字符数 | 50,000 字符 |
