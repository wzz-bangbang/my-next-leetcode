# 用户答案 API

> 路径：`/api/answers`

**注意**：所有接口需要用户登录

---

## POST - 保存用户答案

**请求**：`POST /api/answers`

**鉴权**：需要登录

**请求体**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `questionId` | number | 是 | 题目ID |
| `code` | string | 是 | 用户代码 |

**请求示例**：
```json
{
  "questionId": 1,
  "code": "function promiseAll(promises) {\n  // 实现\n}"
}
```

**返回字段**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `success` | boolean | 是否成功 |

**返回示例**：
```json
{
  "success": true
}
```

---

## 错误响应

| 状态码 | 说明 |
|--------|------|
| 400 | 参数错误（缺少 questionId 或 code） |
| 401 | 未登录 |
| 500 | 数据库错误 |

---

## 前端限制

| 限制项 | 值 |
|--------|-----|
| 最大行数 | 400 行 |
| 最大字符数 | 50,000 字符 |

---

## 备注

用户保存的代码通过 `/api/code?id=xxx` 题目详情接口的 `savedCode` 字段返回，不需要单独调用 GET 接口。
